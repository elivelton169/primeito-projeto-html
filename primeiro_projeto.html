Estrutura Básica do HTML

(Voltar)

A estrutura de uma página em HTML ela consiste em 3 partes sendo: o hmtl, a cabeçalho (head) e o corpo (body):

  1. Head é a parte que não é exibida no navegador da Web quando a página é carregada, porém ela recebe uma tag (Title) que indica o titulo da página.
  2. Body é a tag usada para definir o corpo do documento e dentro dela serão colocadas as tags que irão compor a nossa página Web.

Listas Ordenadas e Não Ordenadas

(Voltar)

A principal diferença entre a lista ordenada e a lista não ordenada está na maneira com que os itens de cada lista são apresentados, a lista ordenada enumera os elementos e a lista não ordenada coloca marcadores antes de cada elemento.

  • Lista Ordenadas representa uma lista de itens ordenados que forma característica entre itens ordenados em uma lista, são mostrados com uma contagem que os precede ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos.
  • Listas Não Ordenadas define uma lista de itens em que a ordem dos itens não define. Ou em outras palavras, a tag de lista não ordenada é usada para criar uma lista não ordenada.
